# Rate-Monitor
This project mainly grab the GBP/CNY data from Bank of China, and set a monitor to push Exchange Rate message by Wechat
First of all, we need a tool called PUSHPLUS which is a Wechat official accounts can send editional messages from server. we can enter the Token into the code.
Then,configure Exchange rate name variables like 英镑/美元
after that, set the rate alarming level and check_interveral.
